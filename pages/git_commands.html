Basics Git Commands

CLI:

Clone a Repo

git clone repo name

Print Current Repo Information

git remote -v

Add a Change

git add file name

Show Git Status

git status

Commit a Change

git commit -m"commit message"

Show Git Logs

git log

Update Repository to GitHub

git push origin main

Conclusion

By now you should be able to:

  • Maintain your git hub repository through:
    • git clone
    • git push
  • Stage and commit code changes with:
    • git add
    • git commit -m
  • Monitor the status of your commits:
    • git log
    • git status
